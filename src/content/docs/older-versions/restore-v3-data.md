When Session Buddy upgrades from v3 to v4 it normally automatically transfers your data. However, because of Chrome deprecations, Session Buddy will be unable to do this automatically after Chrome v124.

If you miss the auto-migration window, you may be able to restore your Session Buddy v3 data manually. To do so, use the [Session Buddy v4 migration tool](/v4-migration/).

This tool walks you through a few steps to create a restore file that can be dropped into Session Buddy v4 using the [Restore feature](/restore/).
