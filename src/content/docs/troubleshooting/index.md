This article outlines the first steps to take if you encounter a problem while using Session Buddy.

## 1. Save your data

Begin by backing up your Session Buddy data:

- **Using Session Buddy's Backup Feature:** Click the gear icon in Session Buddy and select "Backup" to save your collections and history. See [Backup and Restore](/backup-restore/).
- **If Session Buddy Won't Open:** Use the `saveData()` [console command](/console/#console-commands) to download a file containing all collections and history.
- **Manual Backup:** If the above methods fail, manually copy the [Session Buddy data folders](/data-location-v4/) to a safe location.

## 2. Restart the browser

Ensure a complete shutdown of your browser before restarting.

<img src="/src/assets/images/windows.png" alt="Windows" style="display:inline;vertical-align:baseline;margin-right:8px;transform:translateY(3px);width:18px;" /> **On Windows:** Close all browser windows. Then check the tool tray for the browser icon. If it is present, right-click it and click "Exit".

<img src="/src/assets/images/apple.png" alt="macOS" style="display:inline;vertical-align:baseline;margin-right:10px;transform:translateY(3px);width:16px;" /> **On macOS:** Press `Cmd+Q` to quit the browser. Then check the Dock for the browser icon. If it's still there, right-click and choose "Quit" if available.

After restarting, check if the issue persists. If it does, proceed to the next step.

## 3. Check the docs

Visit the [Session Buddy Docs](/docs) for solutions to common issues. It's likely you'll find the assistance you need there.

If you're unable to find an answer, [contact support](/contacting-support/).
