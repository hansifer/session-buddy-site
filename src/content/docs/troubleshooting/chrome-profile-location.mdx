Chrome allows for multiple [user profiles](https://support.google.com/chrome/answer/2364824?hl=en).

If you don't have multiple profiles set up in Chrome, then your settings and extensions are tied to the **Default** profile.

To locate the folder of the current Chrome profile, do the following:

1. Copy and paste the following URL into a new tab and hit Enter:

<p></p>

`chrome://version/` <CopyButton content="chrome://version/" className="ml-2" client:load />

This page displays information about your Chrome installation.

2. Select and copy the full value of the **Profile Path** displayed on that page.

> <img src={windowsImg.src} alt="Windows" style="display:inline;vertical-align:baseline;margin-right:8px;transform:translateY(3px);width:18px;" /> **On Windows**, it looks something like this:
>
> <code>C:\Users\\<span style="color:#386296;">username</span>\AppData\Local\Google\Chrome\User Data\\<span style="color:#386296;">profileName</span></code>

> <img src={appleImg.src} alt="macOS" style="display:inline;vertical-align:baseline;margin-right:10px;transform:translateY(3px);width:16px;" /> **On a Mac**, it looks something like this:
>
> <code>/Users/<span style="color:#386296;">username</span>/Library/Application Support/Google/Chrome/<span style="color:#386296;">profileName</span></code>

3. Go to the profile path copied in the previous step.

> <img src={windowsImg.src} alt="Windows" style="display:inline;vertical-align:baseline;margin-right:8px;transform:translateY(3px);width:18px;" /> **On Windows**: paste the path into the File Explorer address bar and hit Enter.

> <img src={appleImg.src} alt="macOS" style="display:inline;vertical-align:baseline;margin-right:10px;transform:translateY(3px);width:16px;" /> **On a Mac**: select Go in the Finder menu, then click Go to Folder. Paste the path into the textbox and click Go.
